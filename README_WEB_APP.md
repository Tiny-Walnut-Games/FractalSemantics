# FractalSemantics Experiment Orchestrator - Web Application

## Overview

The FractalSemantics Experiment Orchestrator is an educational web application that provides interactive access to all 12 FractalSemantics validation experiments with comprehensive step-by-step mathematical walkthroughs. This application transforms complex computational concepts into accessible educational experiences.

## Quick Start

### 1. Launch the Application

Simply open `index.html` in any modern web browser:

```bash
# Open in default browser
open index.html  # macOS
start index.html  # Windows
xdg-open index.html  # Linux
```

### 2. Explore Experiments

- **Individual Experiments**: Click "Run Experiment" on any experiment card
- **Quick Test Suite**: Run core experiments (EXP-01 through EXP-04) for rapid validation
- **Full Suite**: Execute all 12 experiments sequentially

### 3. Educational Content

- **Mathematical Concepts**: Click "Show Math" to view key formulas and concepts
- **Step-by-Step Process**: Click "Show Steps" to see detailed procedures
- **Live Output**: Watch real-time execution with educational commentary
- **Results Analysis**: View comprehensive summaries and key learnings
